== Description ==

Replace words or phrases in your content and manage the links 
settings in all your wordpress apps in one central manage console.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload folder `content-link-center` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Get the API key from `http://www.def.ch/content-link-center/`. If you haven't an account, create now for free
4. Save the API key in your `Content Link Center` plugin

== Upgrade Notice ==
